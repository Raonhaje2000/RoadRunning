## Road Running

길에 있는 장애물에 부딪히지 않도록 조심하면서 맵 끝에 도달하는 게임입니다.

오른쪽 상단의 스톱워치를 통해 도착까지 걸린 시간을 측정할 수 있습니다.

방향키를 사용해 캐릭터를 움직일 수 있으며, 왼쪽 Ctrl키로 캐릭터를 멈출 수 있습니다.
